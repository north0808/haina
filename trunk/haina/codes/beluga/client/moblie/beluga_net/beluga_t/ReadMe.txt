========================================================================
    WIN32 应用程序：beluga_t 项目概述
========================================================================

应用程序向导已为您创建了此 beluga_t 应用程序。  

此文件概述了组成 beluga_t 应用程序的每个文件
的内容。


beluga_t.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件。 
    它包含有关生成该文件所使用的 Visual C++ 版本的信息，以及
    有关在应用程序向导中选择的平台、配置和项目功能
    的信息。

beluga_t.cpp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
应用程序向导创建了下列资源：


beluga_tppc.rc
    这是程序使用的所有 Microsoft Windows 资源的清单
    （当针对 Pocket PC 2003 平台或
    支持相同的用户界面模型的平台进行编译时）。它包括 RES 子目录中
    存储的图标、位图和光标。此文件可以
    直接在 Microsoft Visual C++ 中进行编辑。如果 .rc 文件保持不变，
    数据节中的定义将保持为
    它们所定义为的数值的十六进制版本，而不是定义的
    友好名称。

beluga_tppc.rc2
    此文件包含不由 Microsoft 
    Visual C++ 编辑的资源。您应将所有无法由资源编辑器编辑的资源
    放置到此文件中。

Resourceppc.h
    这是标准头文件，它定义新的资源 ID。
    Microsoft Visual C++ 读取并更新此文件。




beluga_t.ico
    这是用作应用程序图标 (32x32) 的图标文件。
    此图标包括在主资源文件 beluga_t.rc 中。

/////////////////////////////////////////////////////////////////////////////
其他标准文件：

StdAfx.h，StdAfx.cpp
    这些文件用于生成名为 beluga_t.pch 的预编译头 (PCH) 文件
    和名为 StdAfx.obj 的预编译类型文件。

其他注释：

应用程序向导使用“TODO:”注释指示
应添加或自定义的源代码部分。
